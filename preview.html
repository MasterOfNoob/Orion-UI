Import Orion UI

            
Add these lines to your HTML file to include the Orion UI library from CDN:

            
<!-- CSS Library -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/Masterofnoob/Orion-UI@master/css/orion.css">

<!-- JS Library -->
<script src="https://cdn.jsdelivr.net/gh/Masterofnoob/Orion-UI/js/orion.js"></script>

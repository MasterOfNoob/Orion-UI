Orion UI

A CSS & JS library for building stellar interfaces.

Library Components

A collection of galactic UI elements for your next project. Click on a component's header to jump to its code snippet.

Matrix Grid

An animated, data-driven grid with cascading binary code. Perfect for dashboards and terminals.

<div id="orion-matrix-grid"></div>

Code Terminal

A customizable terminal that simulates typing a sequence of commands with a blinking cursor.


                    

                

                
<div class="orion-terminal">...</div>

            


            

                
Buttons

                
Use the .orion-btn class with a style variant. The .orion-ripple class adds a click animation.

                

                    
                    
                

                
<button class="orion-btn orion-btn-primary">...</button>

            


            

                
Form Elements

                
Minimalist input fields and text areas with a glowing focus state.

                

                    
                    
                

                
<input type="text" class="orion-input">

            


            

                
Cards

                
The .orion-card class creates a semi-transparent container with a subtle shadow and hover effect.

                

                    

                        
This is a nested card. Cards are great for data displays.

                    

                

                
<div class="orion-card"></div>

            


            

                
Progress Bar

                
A loading bar with a subtle animated shimmer. Set the progress with the data-value attribute.

                

                    

                        

                    

                

                
<div class="orion-progress-bar" data-value="65"></div>
